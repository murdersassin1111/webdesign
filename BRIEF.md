# Design Principles (from Refactoring UI)
- Use 4/8/16/24/32/48/64px spacing scale only
- Max 2 fonts: one for headings (weight 700-800), one for body (weight 400-500)
- Limit palette: 1 brand colour + neutrals + 1 accent
- Avoid pure black (#000) - use dark saturated neutrals (#1a1a2e, #0f172a etc)
- Hierarchy via weight + size + colour, not just size
- Fewer borders, more whitespace and shadows
- Mobile-first always
